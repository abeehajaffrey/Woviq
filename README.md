1. Password Strength Checker:

2. Description
This is a Python program that checks the strength of a password entered by the user. It tells whether the password is Weak, Medium, or Strong — based on length, presence of numbers, special characters, and uppercase/lowercase letters.

3. Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks for numbers and special characters
- Rates the password as Weak/Medium/Strong

4. Technologies Used
- Python

5. How to Run
```bash
# Clone the repository
git clone https://github.com/Abeeha17j/password-strength-checker.git

# Go into the folder
cd password-strength-checker

# Run the program
python password_checker.py
```
